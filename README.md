# 🎬 CineFavoritos

Um site pessoal mostrando uma seleção de filmes favoritos — feito com **HTML, CSS e JavaScript puros**, sem frameworks.

🌐 **Acesse o site online:** [jaog1v1.github.io/cinefavoritos](https://jaog1v1.github.io/cinefavoritos/)

## ✨ Funcionalidades

- **Vitrine de 12 filmes** com pôsteres, descrições e nota IMDb
- **Filtro por gênero**: Ação, Ficção Científica e Animação
- **Saudação dinâmica** baseada no horário do dia
- **Página de contato** com validação de formulário em JavaScript
- **Design cinematográfico** com tema escuro, dourado e vermelho
- **Totalmente responsivo** (desktop, tablet e mobile)
- **Animações suaves** de entrada e hover nos cards

## 📁 Estrutura

```
.
├── index.html      # Página principal com a vitrine de filmes
├── contato.html    # Página de contato
├── style.css       # Estilos e responsividade
├── script.js       # Filtro, validação e saudação
└── imagens/        # (Pasta opcional para imagens locais)
```

## 🚀 Como usar

Basta abrir o `index.html` no navegador — não há dependências nem build.
